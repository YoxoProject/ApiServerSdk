L'API utilise le flux OAuth2 Client Credentials. Pour vous authentifier :
1. Obtenez vos identifiants client (client_id et client_secret) depuis le serveur d'authentification de Yoxo
2. Demandez un jeton d'accès : `POST https://auth.yoxo.software/oauth2/token`
   ```
   grant_type=client_credentials
   &client_id=votre_client_id
   &client_secret=votre_client_secret
   &scope=api_access
   ```
3. Incluez le jeton dans l'en-tête Authorization : `Bearer <token>`
