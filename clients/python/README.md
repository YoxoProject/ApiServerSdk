2. Demandez un jeton d'accès : `POST https://auth.yoxo.software/oauth2/token`

   ```
   &client_id=votre_client_id
   &client_secret=votre_client_secret
   &scope=api_access
   ```
3. Incluez le jeton dans l'en-tête Authorization : `Bearer <token>`
